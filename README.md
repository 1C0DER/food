# 🍲 Local Bites

## 📌 Overview
Local Bites is a web application designed to showcase African (specifically Nigerian) cuisine.  
It helps users discover traditional dishes, learn how to cook them, and find places where they can purchase authentic meals.

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, React  
- Backend: PHP / Python  
- Database: MySQL  
- APIs: External APIs (restaurant data)  
- Design: Figma  

---

## Core Features

### 🔹 Food Discovery
- Browse different categories of Nigerian dishes  
- “Popular” section to highlight trending foods  
- Detailed food cards including:
  - Ratings  
  - Short history  
  - Description  

---

### 🔹 Recipes Section
- Daily featured recipes  
- Organized by categories (easy navigation)  
- Includes:
  - Ingredients  
  - Cooking guides  
  - Tutorial videos  

---

### 🔹 Blog System
- Displays:
  - Most popular blogs  
  - Latest posts  
- Structured layout for easy reading  
- Encourages users to share knowledge  

---

### 🔹 User Interaction
- Sign up / login system  
- Users can:
  - Share recipes  
  - Share restaurant info  
- Call-to-action banners across pages  

---

### 🔹 Ordering Option
- Option to purchase dishes (via external integrations)
